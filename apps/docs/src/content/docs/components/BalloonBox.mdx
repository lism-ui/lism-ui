## Props

| プロパティ | 説明 |
|---|---|
|`variant`| ふきだしの方向を指定できます。`c--balloonBox--{variant}` クラスが出力されます。|
|`keycol`| ふきだしの色を指定できます。`u--colbox`も出力されます。|
