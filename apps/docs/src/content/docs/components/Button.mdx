## Props

| プロパティ | 説明 |
|---|---|
|`variant`| `c--button--{variant}` クラスを出力します。`fill`,`outline`の2種類のスタイルをこの例では用意しています。|
{/* |`layout`| Buttonを構成するレイアウト用コンポーネントを指定できます。デフォルトは`Lism`です。| */}
{/* |`icon`, `endIcon` | ボタンテキストの左右に表示するアイコンを指定できます。| */}
{/* |`hover`| デフォルトは`fade`が指定されています。| */}


{/* ## 注意点
Common Props の `w`, `h` ではなく、`width`, `height` を使ってください。 */}



## Usage

{/* ### Buttonのvariantスタイル */}
{/* (`variant`を指定しなければ`'fill'`になります。) */}
{/* カラー指定は`keycol`を使います。 */}

`variant='fill'`は`bgc`,`variant='outline'`は`c`でカラーを指定します。  
（`variant`がない場合、`c`,`bgc`,`bdc`で自由に指定してください。）

## Opt-in

### `hov="reverse"`

ホバー時に、fill,outlineが入れ替わるような [`-hov:`クラス](/ja/css/hover)を追加する例を紹介します。


<PreviewTitle>次のようなCSSを追加してください。</PreviewTitle>
```scss
.-hov\:reverse {
	--hov-c: var(--bgc);
	--hov-bgc: var(--c);
	&:where(.c--button--fill) {
		--hov-bgc: transparent;
	}
	&:where(.c--button--outline) {
		--hov-c: var(--c--base);
	}
	@media (any-hover: hover) {
		&:hover {
			background-color: var(--hov-bgc) !important;
			color: var(--hov-c) !important;
		}
	}
}
```
