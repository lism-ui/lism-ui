## Import

<ImportSource name='Callout' react astro others={['presets.js']}/>


## Props

| プロパティ | 説明 |
|---|---|
|`type`|`keycol`と`icon`の組み合わせを呼び出すキーワードを指定できます。初期値:`"alert"`|
|`icon`| アイコンを指定できます。デフォルトは `"alert"`です。|
|`keycol`| キーカラーを指定します。|
|`isFlow`|`__body`に渡されます|
{/* | `iconProps` | アイコン（`<Icon>`）に渡すpropsを指定できます。| */}
{/* - `variant`: `'fill'` | `'outline'` ? */}
