## Props

| プロパティ | 説明 |
|---|---|
|`layout`| 展開するレイアウトコンポーネントを指定します。デフォルトは`<WihSide>`です。|
|`href`| この例では、hrefを指定すると自動で`a.is--linkBox`として展開されるようになっています。|

### カードにリンクをつける
