ダミーテキストを出力できるLismコンポーネントです。


## Props

| プロパティ | 説明 |
|---|---|
| `lang` | 言語を指定できます。デフォルトで`en`。そのほか、`ja`をサポートしています。 |
| `length` | ダミーテキストの長さを指定できます。有効な値は `xs`, `s`, `l`, `xl` です。 |
| `pre` | ダミーテキストの前に表示するテキストを指定できます。 |
| `tag` | ダミーテキストのタグを指定できます。 |
| `offset` | ダミーテキストのオフセットを指定できます。`,`、`.`単位で区切って切り捨てします。（長さは短くなります）   |
