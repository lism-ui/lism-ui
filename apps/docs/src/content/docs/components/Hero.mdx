## Usage

{/* `HeroContent` には`type`を指定でき、`top`、`center`、`bottom`のいずれかを指定することができます。  
それぞれのキーワードに合わせて出力されるpropsが変わります。 */}

{/* たとえば、`type='center'`の場合は、`my='auto'`, `hasGutter`, `isContainer='m'`が呼び出されます。 */}

{/* <Stack isFullwide ar='2/1' bgc='base-2' py='50'>
	<Container size='m' hasGutter isFlow my='auto'>
		<Text fw='bold' fz='xl'>DEMO 01</Text>
		<Dummy length='xl' lang='ja'/>
	</Container>
</Stack> */}
