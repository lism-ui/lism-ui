## Props

| プロパティ | 説明 |
|---|---|
|`size`| アバターのサイズを指定します。|
|`src`| アバター画像のパス |
|`alt`| altテキスト |


{/* |`mediaProps`| 上記以外にメディア側に対してプロパティを指定したい場合はこのプロパティにオブジェクト形式で指定できます。| */}
