アバター画像用のコンポーネントです。


## Props

| プロパティ | 説明 |
|---|---|
|`size`| アバターのサイズを指定します。|
|`name`| アバターの名前を指定します。`alt`属性に出力されます。|
|`src`| `src`が指定されていれば子要素にメディアを表示します。(`children`がある場合は無視されます。)|
|`mediaProps`| 上記以外にメディア側に対してプロパティを指定したい場合はこのプロパティにオブジェクト形式で指定できます。|
