アバター画像用のコンポーネントです。


## 構造
```pug
div.b--avatar.is--frame(--size)
	img
```


## Props

| プロパティ | 説明 |
|---|---|
|`variant`| `b--avatar--{variant}` クラスを出力します。|
|`size`| アバターのサイズを指定します。|
|`src`| `src`が指定されていれば子要素にメディアを表示します。(`children`がある場合は無視されます。)|
{/* |`mediaProps`| 上記以外にメディア側に対してプロパティを指定したい場合はこのプロパティにオブジェクト形式で指定できます。| */}

{/* Memo: placeholder`で画像じゃなくて一文字とかで表示できるようにしたい */}
