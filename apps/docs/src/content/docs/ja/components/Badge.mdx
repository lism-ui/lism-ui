バッジを表示するコンポーネントです。

## 構造
```pug
span.b--badge
```

## Props

| プロパティ | 説明 |
|---|---|
|`variant`| `b--badge--{variant}` クラスを出力します。`fill`を標準で用意しています。|


## Import

<ImportPackage component='Badge' />

## Usage

export const ColorDemos = ({variant = '',  isColored}) => {
	const badgeProps = {variant, bdrs:'1'};

	return (
		<Flex variant='cluster' g='20'>
			{['red', 'orange', 'yellow', 'green', 'blue', 'purple', 'pink', 'gray'].map((_colr) => {
				if(variant === 'fill') badgeProps.bgc = _colr;
				if(variant === 'outline') badgeProps.c = _colr;
				if(isColored) { badgeProps.isColored = true; badgeProps.keycolor = _colr; }

				return (
					<Badge key={_colr}  {...badgeProps}>{_colr}</Badge>
				);
			})}
		</Flex>
	);
}

### アイコンを使用する例

`<Badge>`に`icon`を指定すると、アイコンを出力できます。  
この時、内部で`<Icon variant='inline'>`が呼び出されます。

<Callout type='info'>
`.b--badge`には`display: inline-flex;`がデフォルトで指定されています。
</Callout>
