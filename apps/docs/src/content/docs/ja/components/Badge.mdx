バッジを表示するコンポーネントです。

## 構造
```pug
span.b--badge.-boxcolor:
```

## Props

| プロパティ | 説明 |
|---|---|
|`variant`| `b--badge--{variant}` クラスを出力します。`fill`, `outline` の2種類を標準で用意しています。|
|`boxcolor`| バッジのカラー。[`COLOR`トークン](/ja/css/tokens/#color)の他、任意のカラーコードを指定できます。|

<ImportCode component='Badge' />

## Usage

export const ColorDemos = ({colors, variant=''}) => {
	return (
		<Flex variant='cluster' g='20'>
			{colors.map((colorname) => {
				return (
					<Badge key={colorname} boxcolor={colorname} variant={variant} bdrs='xs'>{colorname}</Badge>
				);
			})}

		</Flex>
	);
}


`<Badge>`(`.b--badge`)は、[boxcolor(`-boxcolor:`)](/ja/styles/boxcolor/) でカラーを指定します。

### アイコンを使用する例

`<Badge>`に`icon`を指定すると、アイコンを出力できます。  
この時、`<Icon variant='inline'>`が呼び出されます。
