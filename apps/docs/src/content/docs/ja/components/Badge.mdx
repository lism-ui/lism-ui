バッジを表示するコンポーネントです。


## Props

| プロパティ | 説明 |
|---|---|
|`boxcolor`| バッジのカラー。[`COLOR`トークン](/ja/css/tokens/#color)の他、任意のカラーコードを指定できます。|
|`variant`| 表示スタイル。`fill`, `outline` の2種類を標準で用意しています。|

## Usage

export const ColorDemos = ({colors, variant=''}) => {
	return (
		<Cluster gap='20'>
			{colors.map((colorname) => {
				return (
					<Badge key={colorname} boxcolor={colorname} variant={variant}>{colorname}</Badge>
				);
			})}

		</Cluster>
	);
}


`<Badge>`(`.b--badge`)は、`-boxcolor:` の指定が必須です。  
追加で、 `variant` を指定することでスタイルを変更できます。

### アイコンを使用する例

`<Badge>`に`icon`を指定すると、アイコンを出力できます。  
この時、`<Icon variant='inline'>`が呼び出されます。
