バナーリンクを表示するコンポーネントです。  
`is--frame`, `is--linkBox` が呼び出されます。

## Props

`<Frame>`, `<LinkBox>` で使用できるプロパティが利用できます。

## Import

<Preview>
	<PreviewCode slot='tab' label='React'>
```jsx
import { Banner } from '@lism-ui/core';
```
	</PreviewCode>
	<PreviewCode slot='tab' label='Astro'>
```jsx
import { Banner } from '@lism-ui/astro';
```
	</PreviewCode>
</Preview>
