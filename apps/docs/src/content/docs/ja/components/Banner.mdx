バナーリンク型コンテンツを作成する例を紹介します。

is--frame,is--linkBox を活用します。

## Usage

<Preview>
	<PreviewTitle>16:9のバナーリンク</PreviewTitle>
	<PreviewArea p={30}>
		<LinkBox isFrame href='#banner-link' aspect='16/9'>
			<img src='/img.jpg' alt='' width='960' height='640' loading='lazy' />
		</LinkBox>
	</PreviewArea>
	<PreviewCode slot='tab' label='React'>
```jsx "LinkBox" "isFrame"
import { LinkBox } from '@lism-ui/core';

<LinkBox isFrame href='#banner-link' aspect='16/9'>
	<img src='/img.jpg' alt='' width='960' height='640' loading='lazy' />
</LinkBox>
```
	</PreviewCode>
	<PreviewCode slot='tab' label='HTML'>
```html "is--frame" "is--linkbox"
<a class="is--frame is--linkBox -aspect:16/9 -hov:fade" href="#banner-link">
	<img src="/img.jpg" alt="" width="960" height="640" loading="lazy">
</a>
```
	</PreviewCode>
