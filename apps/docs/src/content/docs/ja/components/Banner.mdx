import { Dummy, Banner, Text, Stack, Center, Layer, Media, Button, LinkBox } from '@lism-ui/astro';
import { Preview, PreviewArea, PreviewTitle, PreviewCode, ImportCode } from '~/components';



バナーリンクを表示するコンポーネントです。  
`<LinkBox layout={Frame}>` が内部で呼び出されます。


## 構造
```pug
a.b--banner.l--frame.is--linkBox
```



## Props

| プロパティ | 説明 |
|---|---|
|`variant`| `b--banner--{variant}` クラスを出力します。|


## Import

<ImportCode component='Banner' />
