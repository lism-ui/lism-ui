バナーリンクを表示するコンポーネントです。  
`<LinkBox isFrame>` が内部で呼び出されます。


## 構造
```pug
a.b--banner.is--linkBox.is--frame
```



## Props

| プロパティ | 説明 |
|---|---|
|`variant`| `b--banner--{variant}` クラスを出力します。|

`<Frame>`, `<LinkBox>` で使用できるプロパティも利用できます。
