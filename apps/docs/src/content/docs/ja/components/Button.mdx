ボタンを表示するコンポーネントです。


## 構造
```pug
a.b--button.b--button--{fill|outline|ghost}.l--flex|grid
	svg.b--button__icon
	span.b--button__text
	svg.b--button__endIcon
```


## Props

| プロパティ | 説明 |
|---|---|
|`variant`| `b--button--{variant}` クラスを出力します。デフォルトで`fill`が指定されます。そのほか、`outline`、`ghost`の2種類のスタイルを用意しています。|
|`keycolor`| ボタンのカラー。[`COLOR`トークン](/ja/css/tokens/#color)の他、任意のカラーコードを指定できます。|
|`layout`| Buttonを構成するレイアウト用コンポーネントを指定できます。デフォルトは`Flex`です。|
|`icon`, `endIcon` | ボタンテキストの左右に表示するアイコンを指定できます。|

{/* |`hover`| デフォルトは`fade`が指定されています。| */}


{/* ## 注意点
Common Props の `w`, `h` ではなく、`width`, `height` を使ってください。 */}

<br />

`variant` が `outline` または `ghost` の時、ホバー時の文字色 が `var(--c--base)`へ変化します。
<Callout type='warning'>
これにより、テキストカラーの白黒が反転してるようなエリアでは、真っ白になったりしてしまうことに注意が必要です。
</Callout>

その場合、`-hov:reverse`は使わず、**ホバー時の色を明示的に指定**して回避してください。
