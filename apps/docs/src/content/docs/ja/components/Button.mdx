ボタンを表示するコンポーネントです。

## Props

| プロパティ | 説明 |
|---|---|
|`keycolor`| ボタンのカラー。[`COLOR`トークン](/ja/css/tokens/#color)の他、任意のカラーコードを指定できます。|
|`variant`| ボタンの表示スタイル。`outline`、`ghost`の2種類を用意しています。|
|<PropBadge>`Button`</PropBadge><br/>`leftIcon`, `rightIcon` | ボタンテキストの左右に表示するアイコンを指定できます。|
|<PropBadge>`Button`</PropBadge><br/>`isGrid`| テキストとアイコンの並びがgridレイアウトになり、アイコンを端に寄せることができます。|
|<PropBadge>`IconButton`</PropBadge><br/>`icon`| `icon`に指定するとアイコンのみ表示される。|
{/* |`hover`| デフォルトは`fade`が指定されています。| */}


{/* <Callout type="info">
Astroでアイコンの受け渡しどうする...。（jsxコンポーネントであれば問題なし。）
</Callout> */}



{/* ## 注意点
Common Props の `w`, `h` ではなく、`width`, `height` を使ってください。 */}

### hov="reverse"

Buttonでは、`-hov:reverse`との相性が良くなるように調整を加えています。

`-hov:reverse`は、基本的には、 `--c`と`--bgc`を入れ替えるようになっていますが、Buttonでは、以下のような挙動になります。

- `b--button--fill` で使うと、背景色が透明になり、文字色がホバー前の背景色になります。さらにボーダーも出現します。
- `b--button--outline`, `b--button--ghost` で使うと、文字色 が `var(--base)`へ変化し, 背景色がホバー前の文字色になります。

		</Cluster>
