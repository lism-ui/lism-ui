## Import
<ImportSource name='Card' react />


## Props

| プロパティ | 説明 |
|---|---|
|`layout`| 展開するレイアウトコンポーネントを指定します。デフォルトは`<WihSide>`です。|


### Child Components

- `<CardMedia>`: `<FlexItem layout={Frame}>`を呼び出します.
- `<CardBody>`: `<FlexItem layout={Stack}>`を呼び出します.

### カードにリンクをつける

`<Card>`に`href`を指定すると、`tag='a'` が適用され、カード全体を`is--linkBox`でリンク化します。
