### カードにリンクをつける

`<Card>`にhrefを指定すると、カード全体が`<LinkBox>`で構成されるようになり、リンクになります。

<Preview>
	<PreviewTitle>例</PreviewTitle>
	<PreviewArea resize p='30'>
		<Card href='###' sideW='50%' >
			<CardMedia isSide aspect='16/9'>
				<Media src='/img/img-1.jpg' alt='' width='960' height='640' />
			</CardMedia>
			<CardBody gap='10'>
				<h3 className='-fz:m'>Card Title</h3>
				<DammyText lang='ja' fz='s' mb='10' />
			</CardBody>
		</Card>
	</PreviewArea>
	<PreviewCode slot='tab' label='Usage'>
		```jsx "href='###'"
		<Card href='###' sideW='50%' >
			<CardMedia isSide aspect='16/9'>
				<Media src='/img/img-1.jpg' alt='' width='960' height='640' />
			</CardMedia>
			<CardBody gap='10'>
				<h3 className='-fz:m'>Card Title</h3>
				<DammyText lang='ja' fz='s' mb='10' />
			</CardBody>
		</Card>
		```
	</PreviewCode>
</Preview>

リンク要素を一部の要素に限定したい場合や、カード全体をリンクにしつつ、別のリンクを含めたい場合、 `is--overlayLink` か `is--hiddenLink` を使用してください。
