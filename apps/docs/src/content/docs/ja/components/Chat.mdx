## Props

| プロパティ | 説明 |
|---|---|
| `avatar` | チャットアバター画像のsrcを指定します。 |
| `name` | チャットに表示する名前を指定します。 |
| `variant` | チャットタイプを指定できます。`speak` または `think` が有効です|
| `direction` | チャットの表示位置を指定します。`start`または`end`を指定します。 |
| `boxcolor` | チャットの背景色を指定します。|
| `footer` | チャット本文の下に表示するフッターテキストを指定できます。 |

### Astroでのslotの活用

Astroでは、name,footerをslotで指定することが可能です。

<Preview>
	<PreviewTitle>`slot`の使用例</PreviewTitle>

		<Chat name='Name A' avatar='/img/avatar01.jpg' boxcolor='green'>
			<i slot='name'>Taro</i>
			<DammyText length='codes' />

		</Chat>
