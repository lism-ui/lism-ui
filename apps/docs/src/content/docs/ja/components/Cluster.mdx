複数の細かい要素をFlexレイアウトで**横並び**に配置します。

`flex-wrap:wrap`がデフォルトで適用されている[`Flex`(`.l--flex`)](/ja/components/flex) の派生のコンポーネントです。


## Props


| プロパティ | 説明 | デフォルト値 |
|---|---|---|
| `itemMinW` <PropBadge type="cssvar">`--item--minW`</PropBadge> | 子要素が最低限維持する幅 | `auto` |


さらに、[`<Flex>`で使えるProps](/ja/components/flex#props)が使えます。
