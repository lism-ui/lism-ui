[`is--container`](/ja/css/design-state/#is--container)をデフォルトで出力するコンポーネントです。

下記の ①, ② はそれぞれ同じ結果になります。
```jsx
// ①
<Lism isContainer='m'>...</Lism>

// ②
<Container size='m'>...</Container>
```


コンテナ要素としての役割をコード上でより明示的にわかりやすくするためにご利用ください。


## Props
| プロパティ | 説明 |
|---|---|
| `size` | コンテナサイズを指定します。|
| `layer` | 展開したい Lismコンポーネント を指定できます。|


## Import

```js
import { Container } from '@lism-ui/core';
```

## Usage

通常のコンポーネントでは`isContainer='m'`のようにしてコンテナサイズを指定しますが、`Container`では`size='m'`のように書くことができます。
