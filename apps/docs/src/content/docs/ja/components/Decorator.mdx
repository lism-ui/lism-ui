コンテンツを装飾するための空要素を出力するためのコンポーネントです。



## Props

| プロパティ | 説明 | デフォルト値 |
|---|---|---|
|`size`| デコレーターのサイズ（横幅・高さを一括指定します。|`1em`|



## Import

<Preview>
	<PreviewCode slot='tab' label='React'>
```jsx
import { Decorator } from '@lism-ui/core';
```
	</PreviewCode>
	<PreviewCode slot='tab' label='Astro'>
```jsx
import { Decorator } from '@lism-ui/astro';
```
	</PreviewCode>
</Preview>


## Usage

	<PreviewTitle>Call</PreviewTitle>

			<Decorator
				size='1.25em'
				bd='l'
				rotate='45deg'
				transformOrigin='left bottom'
			/>
			<Box p='box:s'>
				<DammyText length='m' />
			</Box>
			<Decorator
