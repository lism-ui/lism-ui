コンテンツを装飾するための空要素を出力するためのコンポーネントです。  
デフォルトで`aria-hidden="true"`で出力されます。


## 構造
```pug
div.a--decorator[aria-hidden="true"]
```



## Props

| プロパティ | 説明 |
|---|---|
|`variant`| `a--decorator--{variant}` クラスを出力します。|
|`size`| デコレーターのサイズ（横幅・高さ）を一括指定します。<br/>この指定がある場合、`ar='1'`が自動で付与され、指定したサイズは`w`に渡されます。|
