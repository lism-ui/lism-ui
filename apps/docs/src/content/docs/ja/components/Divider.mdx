区切り線を描画するコンポーネントです。

{/* 縦向き・横向きをborderではなく */}

## 構造
```pug
div.a--divider
```

## Props

| プロパティ | 説明 |
|---|---|
|`variant`| `a--divider--{variant}` クラスを出力します。|
| `bdw`,`bds`,`bdc` | 区切り線のボーダープロパティ |
