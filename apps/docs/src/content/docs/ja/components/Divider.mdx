区切り線を描画するコンポーネントです。

{/* 縦向き・横向きをborderではなく */}

## 構造
```pug
div.l--divider.-bd:[bs|is]
```

## Props

| プロパティ | 説明 |
|---|---|
| `bdw`,`bds`,`bdc` | 区切り線のボーダープロパティ |

## Import

<ImportPackage component='Divider' />
