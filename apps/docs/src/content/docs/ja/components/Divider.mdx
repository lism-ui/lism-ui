区切り線を描画するコンポーネントです。

{/* 縦向き・横向きをborderではなく */}

## Props


| プロパティ | 説明 |
|---|---|
| `bdw`,`bds`,`bdc` | 区切り線のボーダープロパティ |

## Import

<Preview>
	<PreviewCode slot='tab' label='React'>

	</PreviewCode>
	<PreviewCode slot='tab' label='Astro'>

	</PreviewCode>
</Preview>
