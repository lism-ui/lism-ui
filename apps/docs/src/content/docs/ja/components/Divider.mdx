区切り線を描画するコンポーネントです。

{/* 縦向き・横向きをborderではなく */}

## Props


| プロパティ | 説明 |
|---|---|
| `label` | 区切り線の間に表示するラベルテキスト |
| `isVertical` | 区切り線を縦方向にできます |
| `bdw`,`bds`,`bdc` | 区切り線のボーダープロパティ |
