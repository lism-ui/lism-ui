## Import
<ImportSource name='DividerLabel' react astro/>



## Props

| プロパティ | 説明 |
|---|---|
|`textPosition`|ラベルテキストの位置。`start`,`end`を指定するとその方向にテキストが寄ります|
| `bd`,`bdw`,`bds`,`bdc` | 区切り線のボーダープロパティ。 |
