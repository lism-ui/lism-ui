# Frame

[`is--frame`](/ja/css/layout-state/#is--frame)をデフォルトで出力するコンポーネントです。(`<Lism isFrame>`のエイリアスです。)

アスペクト比や高さを指定し、直下のメディア要素（`img`,`vieo`,`picture`,`iframe`）を `object-fit:cover` でFrameサイズいっぱいに表示することができます。


## Props

| プロパティ | 説明 |
|---|---|
| `aspect` | フレームの比率を指定できます。|
| `layout` | 展開したい Lismコンポーネント を指定できます。|
{/* | `objectPosition` | 直下のメディア要素のobject-positionを指定できます。 | */}

<HelpText mbs="em10">(aspect自体はどのコンポーネントにも指定できる CSS Props です。)</HelpText>

### 数値以外での比率指定

- `gold`: 黄金比
- `silver`: 白銀比
- `bronze`: 青銅比
- `ogp`: OG Imageの比率
- `cinema`: シネマ比率

export const FrameRatios = () => {
	return (
		<Columns cols={[1, 2]} g={20} >
			{['gold', 'silver', 'bronze', 'ogp'].map((ratio) => {
				return (
					<Frame key={ratio} aspect={ratio} bgc='base-2'>
						<Text pos='a' t='50' l='50' fz='s' css={{translate:'-50XY'}}>
							<code>{ratio}</code>
						</Text>
					</Frame>
				);
			})}
		</Columns>
	);
}

### Flexの子要素として使用する場合の注意点

Flexの子要素でFrameを使用する場合、 `align-items:stretch;`の影響に注意してください。

他の兄弟要素の高さの方が高い場合、そちらと同じ高さになってしまうため、それを回避したい場合は`align-items`を調整するか、Frame要素をさらにBoxで囲むなどの対処が必要です。

				<DammyText length='l' p='box' />
			</Box>

### aspect はどのコンポーネントにも指定できます

`is--frame` は直下のメディア要素に影響を与えてしまいます。

単純にアスペクト比率を固定したエリア内にコンテンツを配置したいというケースでは、`Frame`を使う必要はありません。  
`aspect`は`Box`など他のコンポーネントでも指定できるので、次のような書き方ができます。

<Preview>
	<PreviewTitle>
		Boxにaspectを指定する例
	</PreviewTitle>
	<PreviewArea resize p='5'>
		<Box aspect='2/1' ovy='auto' isFlow p='box:l' bgc='base'>
			<DammyText />
			<DammyText length="l" />
			<Frame aspect='2/1' maxW='50%'>
				<Media src='/img/img-1.jpg' alt='' width='960' height='640' />
			</Frame>
			<DammyText length="xl" />
			<DammyText />
		</Box>
	</PreviewArea>
	<PreviewCode slot='tab' label='JSX'>
		```jsx "Box aspect='2/1'" "ovy='auto'"
		<Box aspect='2/1' ovy='auto' isFlow p='box:l'>
			...
		</Box>
		```
	</PreviewCode>
</Preview>
