# Frame

アスペクト比や高さを指定した時、コンテンツがその高さを超えないように制限します。

また、直下のメディア要素（`img`,`vieo`,`picture`,`iframe`）が `object-fit:cover` でサイズいっぱいに表示されるようになっています。

{/* Layerを配置しやすいように position:relative; を指定しています。 */}


```css
.l--frame {
	overflow: hidden;

	> :where(img, video, picture, iframe) {
		display: block;
		width: 100%;
		height: 100%;
		object-fit: cover;
	}
}
```


## Props

| プロパティ | 説明 |
|---|---|
| `ar` | フレームの比率を指定できます。|
| `layout` | 展開したい Lismコンポーネント を指定できます。|
{/* | `objectPosition` | 直下のメディア要素のobject-positionを指定できます。 | */}

<HelpText>(`ar` 自体はどのコンポーネントにも指定できる CSS Props です。)</HelpText>

### 数値以外での比率指定

ユーティリティクラスをカスタムで追加して利用する例を紹介します。

- `golden`: 黄金比
- `silver`: 白銀比
- `bronze`: 青銅比
- `ogp`: OG Imageの比率
- `cinema`: シネマ比率

export const FrameRatios = () => {
	return (
		<Columns cols={[1, 2]} g='20'>
			{['golden', 'silver', 'bronze', 'ogp', 'cinema'].map((ratio) => {
				return (
					<Frame key={ratio} ar={`u:${ratio}`} pos='r' bgc='base-2'>
						<Text pos='a' t='50%' l='50%' fz='s' css={{translate:'-50XY'}}>
							<code>{ratio}</code>
						</Text>
					</Frame>
				);
			})}
		</Columns>
	);
}

### ar はどのコンポーネントにも指定できます

`ar`は`Box`など他のコンポーネントでも指定できる汎用的なプロパティです。
