コンテンツを**Gridレイアウトで配置する**ためのコンポーネントです。  
[`<GridItem>`](/ja/components/griditem/)と組み合わせて使うとより便利です。

{/* `<Grid>` は `<Box>`に[`isGrid`](/docs/layout-system/is-grid)を指定した状態のコンポーネントで、`<Box isGrid>`と同等です。 */}

`<Grid>`に使用されるクラス`.l--grid`は、次のようになっており、簡単なレイアウトであれば変数セットだけで調整ができるようにもなっています。

```css
.l--grid {
	--gtc: none;
	--gtr: none;
	--gta: none;
	display: grid;
	grid-template: var(--gtr) / var(--gtc);
	grid-template-areas: var(--gta);
}
```

{/* ただし、HTMLを直接書く場合は次の点に注意してください。 */}

{/* <Callout type='alert'>
`--gt`を指定すると`--gtc`, `--gtr`を上書きする形になってしまうので、基本的には、`--gt`を使うときは`--gtc`, `--gtr`は使わないでください。  
（どうしても両方指定したい場合は、`.-gtc:`、`-gtr:`、`-gta:`クラスも併用してください。）
</Callout> */}


{/* `<Grid>`コンポーネントを使用する際は気にしなくて大丈夫です。 */}

<Callout type='good'>
コンポーネントでは、`gtc`、`gtr`、`gta`は基本的には変数のみが出力されます。
</Callout>


## Import

<ImportPackage component='Grid, GridItem' />


## Props

{/* | プロパティ | 出力 | 説明 |
|---|---|---|
| `variant` | <PropBadge type="attr">`.l--grid--{variant}`</PropBadge> | コンポーネントのバリエーションを指定できます。 | */}


Gridは、[`grid`オブジェクト内で指定できるプロパティ](/ja/props/css-props/#grid--flex) をコンポーネントに直接指定できるようになっています。

|Prop|CSS|
|---|---|
|`gd`|`grid`|
|`gt`|`grid-template`|
|`gta`|`grid-template-gta`|
|`gtc`|`grid-template-gtc`|
|`gtr`|`grid-template-rows`|
