**Gridの子要素**向けのプロパティを扱いやすくするためのコンポーネントです。  


## Props

| プロパティ | 説明 |
|---|---|
|`layout`| 展開したい Lismコンポーネント を指定できます。|




また、本来は [`gridItem`オブジェクト内で指定できるプロパティ](/ja/props/css-props/#grid--flex) をコンポーネントに直接指定できるようになっています。

|Prop|CSS|
|---|---|
|`ga`|`grid-area`|
|`gc`|`grid-column`|
|`gr`|`grid-row`|
|`gcs`|`grid-column-start`|
|`grs`|`grid-row-start`|
|`gce`|`grid-column-end`|
|`gre`|`grid-row-end`|


<PreviewTitle>例えば、以下は同じ出力結果になります。</PreviewTitle>
```jsx
// 1.
<Lism gridItem={{'ga': 'areaName'}}>...</Lism>

// 2.
<GridItem ga='areaName'>...</GridItem>
```
