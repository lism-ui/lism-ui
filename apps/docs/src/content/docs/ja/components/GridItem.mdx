**Gridの子要素**のためのコンポーネントです。  
[`<Grid>`](/ja/components/grid/)と組み合わせて活用してください。


## Props

{/* | プロパティ | 説明 | デフォルト値 |
|---|---|---|
|`as` | 別コンポーネントとして展開できます。 | | */}


また、[`gridItem`オブジェクト内で指定できるプロパティ](/ja/props/css-props/#grid--flex) をコンポーネントに直接指定できるようになっています。

|Prop|CSS|
|---|---|
|`ga`|`grid-area`|
|`gc`|`grid-column`|
|`gr`|`grid-row`|
|`gcs`|`grid-column-start`|
|`grs`|`grid-row-start`|
|`gce`|`grid-column-end`|
|`gre`|`grid-row-end`|


このほか、`aslf`(`align-self`)など。
