[`.is--layer`](/ja/css/layout-state/#is--layer)をデフォルトで出力するコンポーネント(`<Lism isLayer>`のエイリアス）です。

`position:absolute`で要素を親要素サイズに合わせて配置できます。

CSSは以下のようになっており、`inset:0`と`overflow:hidden`も指定された状態で配置されます。

```css
.is--layer {
	position: absolute;
	inset: 0;
	overflow: hidden;
}
```

<Reference>
要素を単純に`position:absolute`で配置したいだけの場合は、`-pos:a`クラスなどを使用することもできます。
参考：[absoluteで配置する方法](/ja/props/others/absolute)
</Reference>


## Props

| プロパティ | 説明 |
|---|---|
| `layout` | 展開したい Lismコンポーネント を指定できます。|
| `blur`,`contrast`,`brightness`,`grayscale`,`hueRotate`,`invert`,`saturate`,`sepia` | これらをまとめて `css.backdropFilter` に受け渡します。|



## Import

```js
import { Layer } from '@lism-ui/core';
```

### backdropFilterの活用

Lismコンポーネントでは、`css.backdropFilter` に指定した値を`backdrop-filter`としてインラインスタイルを出力できるようになっていますが、`<Layer>`コンポーネントでは、この機能をより活用しやすいようになっています。

### メディアレイヤー

`is--frame`と`is--layer`を組み合わせることで、親要素のサイズに合わせたメディアレイヤーを配置することができるので、背景画像として使用することができます。

{/* `MediaLayer` */}
{/* **メディア（画像・動画）を親要素サイズに合わせてabsolute配置する**ための`<Layer>`の拡張コンポーネントです。 */}

{/* また、`<MediaLayer>`は、Propsの処理が`<Layer>`とは少し異なります。 */}
{/* | プロパティ | 説明 |
|---|---|
| `src` | `src`が指定されていれば子要素にメディアを表示します。|
| `media`| `src`が指定されている場合の、そのメディアのHTMLタグ、またはコンポーネントを指定できます。デフォルトは`'img'`です。|
| `width`, `height`,`alt`, `loading`, `decoding`,`hover` | `src`が指定されている場合、これらのプロパティはメディア側に振り分けられます。|
| `mediaProps`|上記以外にメディア側に対してプロパティを指定したい場合はこのプロパティにオブジェクト形式で指定できます。| */}

{/* |`loop`, `muted`, `controls`, `autoplay`, `playsinline`, `poster` | `src`が指定されていて`media`が`'video'`の場合にこれらのプロパティはメディア側に振り分けられます。| */}
