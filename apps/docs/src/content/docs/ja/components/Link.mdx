リンク用に少し特殊な処理が加えられた`<Lism>`コンポーネントです。

- デフォルトで`<a>`タグが出力されます。
- `isOverlayLink`で`is--overlayLink`クラスが出力されます。
- `isHiddenLink`で`is--hiddenLink`クラスが出力されます。
