ボックス全体がリンクになったモジュールです。

## Props


| プロパティ | 説明 |
|---|---|
|`href`|リンク先を指定できます。|
|`layout`|指定したLismコンポーネントでLinkBoxを展開できます。|
|`target`|target属性を指定できます。|
|`rel`|rel属性を指定できます|
|`openNewTab`|trueにすると、target="_blank"とrel="noopener noreferrer"が自動で付与されます。|
|`setHiddenLink`|`true`にすると、LinkBox内部に隠しリンク要素が出力されます。|
|`linkLabel`|`setHiddenLink`を指定した時に、リンクの読み上げテキストを指定できます。|

{/* |`as`|外部コンポーネントでLinkBoxを展開できます。|
|`asProps`|外部コンポーネントに渡すpropsを指定できます。| */}


## Import

```js
import { LinkBox, OverlayLink } from '@lism-ui/core';
```


## Usage

LinkBoxは、次の3通りの使い方ができます。

1. `<LinkBox>`自体を`<a>`タグにする。
2. `<LinkBox>`が`<div>`で、内部に`e--overlayLink`を配置する。
3. `<LinkBox>`が`<div>`で、内部に`e--hiddenLink`を配置する。

<Alert preset='info'>
2, 3 では、LinkBox内部に別のリンクも配置できます。
</Alert>

### 内部に`e--overlayLink`を配置する

LinkBoxに`href`を指定せずに使うと、`<a>`ではなく`<div>`で出力され、  
内部のアンカー要素に`.e--overlayLink`を併用することで`.is--linkBox`全体をリンクにすることができます。

<Alert preset='info'>
この時、LinkBoxの内部に別のリンクを配置できます。
</Alert>

<Note preset='info' isFlow='s'>
`e--overlayLink`にフォーカスした時のアウトラインをLinkBox全体に広げたい場合は、次のようなCSSを追記してください。

```css
@supports selector(:has(*)) {
	.is--linkBox:has(.e--overlayLink:focus-visible) {
		outline: auto 1px;
		outline: auto 1px -webkit-focus-ring-color;
	}
	.e--overlayLink:focus {
		outline: 0;
	}
}
```
</Note>
