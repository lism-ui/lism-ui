ナビメニューリストを作成できるコンポーネントです。

## Props

| プロパティ | 説明 |
|---|---|
| <PropBadge ml='-1px'>`<NavMenu>`</PropBadge><br/>`nestLevel` | メニューのネストレベルを指定します。 |
| <PropBadge ml='-1px'>`<NavMenu>`</PropBadge><br/>`hovBgc` | メニューホバー時の背景色 |
| <PropBadge ml='-1px'>`<NavMenu>`</PropBadge><br/>`hovC` | メニューホバー時の文字色 |
| <PropBadge ml='-1px'>`<NavMenu>`</PropBadge><br/>`passProps.p` | 各メニューのpaddingを一括調整できます |
| <PropBadge ml='-1px'>`<NavMenu>`</PropBadge><br/>`passProps.gap` | 各メニューのgapを一括調整できます |
| <PropBadge ml='-1px'>`<NavMenuItem>`</PropBadge><br/>`href` | NavMenuにhrefを指定すると、NavMenuLinkを省略できます。 |



## Import

```js
import { NavMenu } from '@lism-ui/core';
```


## Usage

export const Items = ({ length = 3, ...props }) => {
	// const array = Array.from({length:3});
	return (
		<>
			{[...Array(length)].map((_,i) => {
				return <NavMenuItem key={i} href={`#menu${i+1}`} {...props}>
					Menu item {i+1}
				</NavMenuItem>;
			})}
		</>
	);

	// return array.map((v,i) => {
	// 	return <p key={i}>aaa</p>;

	// });
};

};

### メニューのネスト

`<NavMenuItem>`に`href`を渡す省略方式ではなく、`<NavMenuLink>`を使ってリンク部分とネストするリストを分けて記述する必要があります。

また、ネストの深さに応じて`nestLevel`を指定してください。
