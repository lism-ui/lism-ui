### Props

| プロパティ | 説明 |
|---|---|
|`keycolor`| 表示カラーを指定します。|
|`icon`| アイコンを指定できます。|
|`type`|`keycolor`と`icon`の組み合わせのプリセット値を指定できます。初期値:`"note"`|
|`title`|タイトルテキストを指定します。|
|`isFlow`|`__body`に渡されます|
{/* - `variant`: `'fill'` | `'outline'` ? */}
