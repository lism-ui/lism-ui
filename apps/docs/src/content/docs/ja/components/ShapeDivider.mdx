## Props

| プロパティ | 説明 |
|---|---|
| `shape` |  形状の指定。svgで描画されます。 `slope`, `wave`, `arrow`, `circle` が指定できます。|
| `isFlip` | シェイプの上下反転をするかどうか。 |
| `level` | 高さのレベル。正の数・負の数を指定できます。 `5` がデフォルト。|
| `stretch` | 横方向の伸縮率。 `1` がデフォルトです。 |
| `offset` | X方向の位置をずらす。 `stretch`と合わせて`%`単位で指定してください。（例: `stretch:1.5` なら、`offset:-25%~25%`の範囲が有効です。） |


- 任意のsvgの受け渡しができるようにしたい<MemoBadge>未実装</MemoBadge>
- `level`について
	- `level * 1cqw` で`<Divider>`の高さが計算され、`600px`時点の値(`level * 6px`)が最小値、`1200px`時点の値(`level * 12px`)が最大値となります。
	- `level` が マイナスの時、図形の描画方向が反転します。
