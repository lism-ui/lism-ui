svgで境界線を描画するコンポーネントです。

## HTML構造

```pug
.b--shapeDivider([data-flip])(--level,--stretch,--offset)
	.b--shapeDivider__inner
		svg.b--shapeDivider__svg
```


## Props

| プロパティ | 説明 |
|---|---|
| `shape` |  形状の指定。svgで描画されます。 `slope`, `wave`, `arrow`, `circle` が指定できます。|
| `shapeFlip` | シェイプの上下反転を指定できます。 `y`で上下反転、`x`で左右反転、`xy`で両方反転です。|
| `level` | 高さのレベルを指定できます。 `5` がデフォルトで、コンポーネントでは正の数・負の数が指定できます。（負の数の時は出力されるsvgに関する処理が加わり、出力される変数はその絶対値になります。）|
| `stretch` | 横方向の伸縮率。 `1` がデフォルトです。 |
| `offset` | X方向の位置をずらす。 `stretch`と合わせて`%`単位で指定してください。（例: `stretch:1.5` なら、`offset:-25%~25%`の範囲が有効です。） |


- `level`について
	- `level * 1cqw` での高さが計算され、`600px`時点の値(`level * 6px`)が最小値、`1200px`時点の値(`level * 12px`)が最大値となります。
	- `level` が マイナスの時、図形の描画方向が反転します。
