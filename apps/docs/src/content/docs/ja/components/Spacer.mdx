import { Spacer, Box, Flex } from '@lism-ui/astro';
import { Preview, PreviewArea, PreviewTitle, PreviewCode, ImportPackage, Callout } from "~/components";


スペースを確保するためのモジュールです。


## 構造
```pug
div.a--spacer[aria-hidden="true"]
```


## Props


| プロパティ | 説明 |
|---|---|
|`variant`| `a--spacer--{variant}` クラスを出力します。|
|`w`| `width`の値|
|`h`| `height`の値|

<Callout type="info">
`<Space>`では、`w`と`h`に対してSPACEトークンの値が使えます。
</Callout>

<ImportPackage component='Spacer' />
