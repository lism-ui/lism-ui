複数の要素をFlexレイアウトで**縦並び**に配置するコンポーネントです。

`<Flex fxd="column">`と機能的には同等です。


## Import

<ImportCode component='Stack' />


## Props

[`<Flex>`で使えるProps](/ja/components/flex#props)が使えます。


## Usage

<Preview>
	<PreviewTitle>シンプルな`<Stack>`の例</PreviewTitle>
	<PreviewArea p='box:l'>
		<Stack g={20} className="has--itemGuide">
			<p>コンテンツ</p>
			<p>コンテンツ</p>
			<p>コンテンツ</p>
		</Stack>
	</PreviewArea>
	<PreviewCode slot="tab" label="JSX">
	```jsx
	<Stack g={20}>
		<p>コンテンツ</p>
		<p>コンテンツ</p>
		<p>コンテンツ</p>
	</Stack>
	```
	</PreviewCode>
	<PreviewCode slot="tab" label="HTML">
	```jsx
	<div class="l--stack -g:20">
		<p>コンテンツ</p>
		<p>コンテンツ</p>
		<p>コンテンツ</p>
	</div>
	```
	</PreviewCode>
</Preview>



### Stackを活用したセンター寄せレイアウト

`<Stack ai="center">`( `.l--stack.-ai:c` )の組み合わせで、Every Layout で言うところの **"内在的な中央寄せ"** （コンテンツが短い時だけ中央寄せ）にできます。

<Preview>
	<PreviewTitle>内在的な中央寄せ</PreviewTitle>
	<PreviewArea resize>
		<Stack ai="center" g={30} p={40} lismState={["has--itemGuide"]}>
			<h3 role="presentation">見出し</h3>
			<DammyText length="s" />
			<DammyText length="l" />
		</Stack>
	</PreviewArea>
	<PreviewCode slot="tab" label="JSX">
	```jsx "ai='center'"
	<Stack ai='center' g={30} p={40}>
		<h3>見出し</h3>
		<p>...</p>
		<p>...</p>
	</Stack>
	```
	</PreviewCode>
	<PreviewCode slot="tab" label="HTML">
	```jsx "-ai:c"
	<div class="l--stack -ai:c -p:40 -g:30">
		<h3>見出し</h3>
		<p>...</p>
		<p>...</p>
	</div>
	```
	</PreviewCode>
</Preview>
