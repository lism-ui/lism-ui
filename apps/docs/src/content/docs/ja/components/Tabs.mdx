タブ要素を作成できるコンポーネントです。  
最低限の実装にしているので、デザインは適宜カスタマイズしてください。


## Props

| プロパティ | 説明 |
|---|---|
|<PropBadge>`<Tabs>`</PropBadge><br/>`variant`|`c--tabs--{variant}`クラスが出力されます。デフォルトは`"default"です。`|
|<PropBadge>`<Tabs>`</PropBadge><br/>`keepHeight`|`data-tabs="keep-height"`属性が出力され、タブの高さを一番コンテンツ量の多いパネルに合わせて固定することができます。`|
|<PropBadge>`<Tabs>`</PropBadge><br/>`isVertical`|`data-tabs="vertical"`属性が出力され、タブ部分が縦並びになります。`|
|<PropBadge>`<TabItem>`</PropBadge><br/>`label` | タブテキストを指定します。 |



{/* - `<Tabs>`
	- `keepHeight` が true の場合、タブの高さを固定する（切り替えた時に高さが動かないようにする）
	- `defaultIndex`で最初に開くタブを指定できる
	- `tabId`: tabID.指定しなければランダム生成されます。(ページ内で一意の値の指定である必要があります。)
- `<TabItem>`
	- `index`: (必須) タブのindex番号 */}

各タブテキストは`<TabItem>``label`で指定できますが、jsx式を使いたい場合、Astroでは`slot='label'`を使ってください。

			<TabItem label='tab2'>
				<DammyText length='l' />
			</TabItem>
		</Tabs>
	</PreviewArea>

### 高さが変わらないようにする

タブを切り替えた時、デフォルトでは、表示されるコンテンツ量が変わることでタブ全体の高さも変わります。  
`<Tabs>`に `keepHeight`(`data-tabs="keep-height"`) を指定すると、タブの高さを一番コンテンツ量の多いパネルに合わせて固定することができます。

<Preview>
	<PreviewTitle>例</PreviewTitle>
	<PreviewArea p={30}>
		<Tabs keepHeight>
			<TabItem label="tab1">
				<DammyText />
			</TabItem>
			<TabItem label="tab2">
				<DammyText length="xl" />
			</TabItem>
		</Tabs>
	</PreviewArea>
	<PreviewCode>
		```jsx "keepHeight"
		<Tabs keepHeight>
			<TabItem label="tab1">
				<p>...</p>
			</TabItem>
			<TabItem label="tab2">
				<p>...</p>
			</TabItem>
		</Tabs>
		```
	</PreviewCode>
</Preview>

	<PreviewCode>
