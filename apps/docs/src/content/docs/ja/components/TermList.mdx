dl,dt,ddタグの組み合わせで、シンプルな説明リストを作成するコンポーネントです。

## Props
- TermList
	- Stack で使えるProps
- TermListItem
	- term : ターム名。(dt要素の内容)
	- dtProps : dtに渡したいProps
	- ddProps : ddに渡したいProps
	- isFlow : dd 側に渡るようになっています。


## Import

```js
import { TermList, TermListItem } from '@loos/lism-core';
```

### SideFlexで展開

TermListItem に isFlex を指定すると、SideFlex で展開され、大きな画面サイズでは横並びになります。
この時、初期値として以下が自動で設定されます。

- gap: 10
- side: 'first'
- sideW: '10rem'
- fluidMinW: '15rem'
