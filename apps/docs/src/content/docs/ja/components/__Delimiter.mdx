import { Text, Cluster, Delimiter } from '@lism-ui/astro';
import { Preview, PreviewArea, PreviewTitle, PreviewCode, ImportCode, PropBadge } from "~/components";


テキストレベルの小さな要素を区切るための要素に使用するコンポーネントです。

デフォルトで`<span>`タグで出力されます。  
また、等幅フォント（<PropBadge type="cssvar">`--ff--mono`</PropBadge>）が適用されるため、記号でデータを区切る際に便利になっています。



## 構造

```pug
span.a--delimiter
```


## Props

| プロパティ | 説明 |
|---|---|
|`variant`| `a--delimiter--{variant}` クラスを出力します。|

{/* が、親に<PropBadge type="cssvar">`--delimiter--c`</PropBadge>を指定することで、デリミタのカラーを一括で変更することができます。 */}
