# Grid

子要素のサイズ比率（`1:2`や`1:1:2`など）を指定して配置できるグリッドモジュールです。

<Preview>
	<PreviewTitle>対応するHTML構造</PreviewTitle>
	<PreviewCode>
	```jsx
	<div class="l--ratioGrid" style="--gtc:{ratio}">...</div>
	```
	</PreviewCode>
</Preview>


## Props

- `ratio`: 要素のサイズ比率を指定します。ブレイクポイント指定可能です。
- その他、[`<Grid>`で使えるprops](/ja/components/grid#props)が使えます。


## Import

```js
import { Grid } from '@loos/lism-core';
```

## Usage
