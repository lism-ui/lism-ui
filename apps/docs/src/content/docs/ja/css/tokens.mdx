### スケーリングについて

Lism では、フォントサイズのスケーリングには数学的に美しいとされるスケーリング理論に基づいて定義することを推奨しています。  
デフォルトでは、**分子8の調和数列**ベースで設計しています。

```scss
:root{
	--fz--5xl: calc(1em * 8 / 2);
	--fz--4xl: calc(1em * 8 / 3);
	--fz--3xl: calc(1em * 8 / 4);
	--fz--2xl: calc(1em * 8 / 5);
	--fz--xl: calc(1em * 8 / 6);
	--fz--l: calc(1em * 8 / 7);
	--fz--m: 1em;
	--fz--s: calc(1em * 8 / 9);
	--fz--xs: calc(1em * 8 / 10);
	--fz--2xs: calc(1em * 8 / 11);
}
```

<Reference>
参考: [音楽、数学、タイポグラフィ](https://standard.shiftbrain.com/blog/music-math-typography)
</Reference>

分子 7~9 あたりで選ぶのがおすすめです。

## `SPACE` トークン

`SPACE`トークンは、次の2種類があります。

|種類|<span class="-whs:nw">変数</span>|説明|
|---|---|---|
|`SPACE.S`|`--s--VAL`|レイアウトの役割としての余白や、コンテンツ間の余白をコントロールするために設計されたトークン。<br />デフォルトでは`rem`単位で、フィボナッチ数に基づいて計算されています。|
|`SPACE.EM`|`--em--VAL`|ボックス内の余白や文字とアイコン要素の間隔など、その要素自体のテキストサイズに合わせて微調整したい時に使用するトークン。`em`単位。|

## `SHADOW` トークン


{/* <Alert preset='warning'>
box-shadow は少し特殊で、ユーティリティクラス内で複数のshadow用変数が組み合わされています。
</Alert> */}

|トークン値|変数|クラス|
|---|---|---|
|`0`|-|`-bxsh:0`|
|`1`|`--sh--1`|`-bxsh:1`|
|`2`|`--sh--2`|`-bxsh:2`|
|`3`|`--sh--3`|`-bxsh:3`|
|`4`|`--sh--4`|`-bxsh:4`|
|`5`|`--sh--5`|`-bxsh:5`|
|`-1`|`--sh--i1`|`-bxsh:-1`|
|`-2`|`--sh--i2`|`-bxsh:-2`|
|`-3`|`--sh--i3`|`-bxsh:-3`|

`--sh--{token}`の値の中身は、次の変数で構成されます。


|構成変数|初期値|
|---|---|
|`--sh-size--0`|`0.5px 1px 2px 0`|
|`--sh-size--1`|`1px 2px 4px 0`|
|`--sh-size--2`|`2px 4px 8px 0`|
|`--sh-size--3`|`4px 8px 16px 0`|
|`--sh-size--4`|`8px 16px 32px 0`|
|`--sh-size--5`|`16px 32px 64px 0`|
|`--sh-hsl`|`215 8% 12%`|
|`--sh-a`|`8%`|
