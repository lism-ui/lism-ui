### `mix` クラスについて

例えば、`.-bgc:mix`クラスは以下のように定義されています。

```css
.-bgc\:mix {
	--bgc1-per: 50%;
	--bgc1: currentColor;
	--bgc2: transparent;
	--bgc: color-mix(in srgb, var(--bgc1) var(--bgc1-per), var(--bgc2));
}
```
