Lismでは、ボックス全体のカラーリングを行う`.-boxcolor:`クラスを用意しています。

| 形式 | 説明 |
|---|---|
| `-boxcolor:` | `--boxcolor`変数と組み合わせて好きなカラーを指定できます。 |
| `-boxcolor:{palette-token}` | パレットトークンに対しては専用のユーティリティクラスが用意されています。 |


<Alert preset='info'>
指定したキーカラーを元に、文字色・背景色・ボーダー色を `color-mix()` で生成します。
</Alert>

## Usage

<Preview>
	<PreviewTitle>例</PreviewTitle>
	<PreviewArea resize p='box:l'>
		<Box boxcolor='blue' p='box'><DammyText length='codes' /></Box>
		<Box boxcolor='red' p='box'><DammyText length='codes' /></Box>
		<Box boxcolor='#688f04' p='box'><DammyText length='codes' /></Box>
	</PreviewArea>
	<PreviewCode slot="tab" label='Component'>
	```jsx
		<Box boxcolor='blue' p='box'>...</Box>
		<Box boxcolor='red' p='box'>...</Box>
		<Box boxcolor='#688f04' p='box'>...</Box>
	```
	</PreviewCode>
	<PreviewCode slot="tab" label="HTML">
	```html
		<div class='l--box -boxcolor:blue -p:box'>...</div>
		<div class='l--box -boxcolor:red -p:box'>...</div>
		<div class='l--box -boxcolor: -p:box' style='--boxcolor:#688f04'>...</div>
	```
	</PreviewCode>
</Preview>

ボーダーカラーは `--bdc`へのセットのみなので`.-boxcolor:`だけでボーダーが表示されるわけではありません。

<Preview>
	<PreviewTitle>ボーダーとの組み合わせ例</PreviewTitle>
	<PreviewArea resize p='box:l'>
		<Box boxcolor='purple' p='box' bd bdw='4px'><DammyText length='codes' /></Box>
		<Box boxcolor='#688f04' p='box' bd><DammyText length='codes' /></Box>
	</PreviewArea>
	<PreviewCode slot="tab" label='Component'>
	```jsx
		<Box boxcolor='purple' p='box' bd bdw='4px'>...</Box>
		<Box boxcolor='#688f04' p='box' bd>...</Box>
	```
	</PreviewCode>
	<PreviewCode slot="tab" label="HTML">
	```html
		<div class='l--box -boxcolor:purple -p:box -bd:' style='--bdw:4px'>...</div>
		<div class='l--box -boxcolor: -p:box -bd:' style='--boxcolor:#688f04'>...</div>
	```
	</PreviewCode>
</Preview>
