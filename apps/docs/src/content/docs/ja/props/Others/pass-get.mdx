複数の子要素へ一括でCSSプロパティを渡すための仕組みを用意しています。

`-get:{prop}`クラスで`--pass--{prop}`変数を受け取るということをルールとします。

<PreviewTitle>例</PreviewTitle>
```css
.-get\:p {
	padding: var(--pass--p);
}
```

コンポーネントを使用する場合は、`passProps`と`getProps`を使います。


<Preview>
	<PreviewTitle>活用例</PreviewTitle>
	<PreviewArea p='20'>

<Spacer h='10'/>
標準で用意されているクラスは以下の通りです。

```css
.-get\:p {padding: var(--pass--p);}
.-get\:m {margin: var(--pass--m);}
.-get\:gap {gap: var(--pass--gap);}
.-get\:c {color: var(--pass--c);}
.-get\:bgc {background-color: var(--pass--bgc);}
.-get\:bdc {--bdc: var(--pass--bdc);}
.-get\:bdw {--bdw: var(--pass--bdw);}
```

足りないものがあれば同じような構造で、自身のプロジェクトのCSSファイルへ追加してください。
