Lismコンポーネントでは、`background`系のpropとして`bg`,`bgc`,`backgrounds`の3種類があります。

| プロパティ | 説明 |
|---|---|
|`bg`| `background` として出力されます。 |
|`bgc`| `background-color` として出力されます。 |
|`backgrounds`| オブジェクトで、background系の各プロパティを指定できます。|


### `bg`

	<PreviewTitle>`bg`に指定</PreviewTitle>
	<PreviewArea resize p={20}>
		<Frame aspect='16/9' bg="url(/img.jpg) center / 50%"></Frame>
	</PreviewArea>
	<PreviewCode>
		```jsx "bg"
		<Frame aspect='16/9' bg="url(/img.jpg) center / 50%"></Frame>
		```
	</PreviewCode>
</Preview>


### `backgrounds`



| キー | 対応するCSSプロパティ |
|---|---|
|image|background-image|
|size|background-size|
|repeat|background-repeat|
|position|background-position|
|color|background-color|
|attachment|background-attachment|
|blendMode|background-blend-mode|
|clip|background-clip|
|origin|background-origin|



<Preview>
	<PreviewTitle>`backgrounds`で指定</PreviewTitle>
	<PreviewArea resize p={20}>
		<Frame
			aspect='16/9'
			backgrounds={{image: 'url(/img.jpg)', position: 'center', size: '50%'}}
		></Frame>
	</PreviewArea>
	<PreviewCode>
		```jsx "backgrounds"
		<Frame
			aspect='16/9'
			backgrounds={{image: 'url(/img.jpg)', position: 'center', size: '50%'}}
		></Frame>
		```
