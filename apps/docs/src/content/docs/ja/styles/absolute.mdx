`position:absolute`で要素を配置する例をいくつか紹介します。

<Note type='info' title='Points'>
- positionにあたるプロパティが`pos`です。
	- `absolute`は`a`と省略できます。
	- `relative`は`r`と省略できます。
- 配置位置を決めるプロパティが `inset`,`t`,`b`,`l`,`r` です。
</Note>

{/* <Callout type='warning' fz='s'>
※ `inset`,`top` `right` `bottom` `left` は、数値指定するとスペーストークンとして認識されます。  
ただし、値が`0%`,`100%`のユーティリティクラス（`.-t:0`,`.-b:100`など）を出力させたい場合、 `t={0}` ではなく `t='0'` と文字列で指定する必要があります。
</Callout> */}
