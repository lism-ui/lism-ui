親要素から複数の子要素に対して同じプロパティを一括で管理したい場合、Lism では `--pass-{prop}`変数を`-pass:{prop}`クラスで直下要素すべてに渡すか、子要素側に`-get:{prop}`クラスをつけて個別に取得する、というシステムを定めています。

			<Box bd>Box</Box>

			<Box bd>Box</Box>

		```jsx "pass" "passVars"
		<Flex variant='cluster' pass='minW,p' passVars={{p: '20 30', minW:'5rem'}} g='20' ta='c'>

			<Box bd>Box</Box>
			<Box bd>Box</Box>
		</Flex>
		```
	</PreviewCode>
	<PreviewCode slot="tab" label="HTML">

同じスタイルを持った子要素の数が多く、HTMLのコード量が冗長的になりすぎる場合に、このシステムを利用してみてください。

ただし、基本的には子要素ひとつひとつに値をセットすればよいことを別アプローチで行なっているだけですので、Lismのコアパッケージで用意しているクラスは最低限に抑えています。

```css
.-get\:c {
	color: var(--pass-c);
}
```

別プロパティに対しても同様のスタイリングを行いたい場合に、同じようなルールでクラスを追加していただくと統一性を保つことができます。
