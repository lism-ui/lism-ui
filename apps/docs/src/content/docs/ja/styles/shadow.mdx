### シャドウカラーの上書き

bxshユーティリティの構成変数である `--sh-color` を上書き指定することで、シャドウカラーを変更することができます。

<Preview>
	<PreviewTitle>`--sh-color`の上書き</PreviewTitle>
	<PreviewArea p={40} nogrid>
		<Box className='is--resetShVar' style={{'--sh-hsl': '20 80% 80%', '--sh-a':'50%'}}>
			<Box bxsh='3' p='30' >...</Box>
		</Box>
	</PreviewArea>
	<PreviewCode slot='tab' label='HTML'>
		```html "--sh-color:"
		<div class="l--box -bxsh:3 -p:30" style="--sh-hsl:20 8% 40%"> … </div>
		```
	</PreviewCode>
	<PreviewCode slot='tab' label='JSX'>
		```jsx "--sh-color:"
		<Box bxsh='3' shColor='rgb(100 100 200 / 20%)' p='30'>...</Box>
		```
	</PreviewCode>
</Preview>
