### シャドウカラーの上書き

`.is--resetShVar` クラスを指定すると、その要素でシャドウ用の各変数が再定義されます。  
この時、シャドウを構成する変数を上書きすることでシャドウの色味などを調整することが可能です。 

<Preview>
	<PreviewTitle>シャドウの上書き例</PreviewTitle>
	<PreviewArea p={40} nogrid>
		<Box className='is--resetShVar' style={{'--sh-hsl': '20 80% 80%', '--sh-a':'50%'}}>
			<Box bxsh='3' p='30' >Box</Box>
		</Box>
	</PreviewArea>
	<PreviewCode slot='tab' label='JSX'>
		```jsx
		<Box className='is--resetShVar' style={{'--sh-hsl': '20 80% 80%', '--sh-a':'50%'}}>
			<Box bxsh='3' p='30' >Box</Box>
		</Box>
		```
	</PreviewCode>
	<PreviewCode slot='tab' label='HTML'>
		```html
		<div class="is--resetShVar l--box" style="--sh-hsl:20 80% 80%;--sh-a:50%"><div class="l--box -bxsh:3 -p:30">Box</div></div>
		```
	</PreviewCode>
</Preview>
