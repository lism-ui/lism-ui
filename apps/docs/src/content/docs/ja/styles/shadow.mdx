### シャドウカラーの上書き

`.is--resetShVar` クラス(仮) を指定すると、その要素でシャドウ用の各変数が再定義されます。  
この時、シャドウを構成する変数を上書きすることでシャドウの色味などを調整することが可能です。 

<Preview>
	<PreviewTitle>シャドウの上書き例</PreviewTitle>
	<PreviewArea p='40' nogrid>
		<Flex className='is--resetShVar -g:40' style={{'--sh-hsl': '200 50% 50%', '--sh-a':'20%'}}>
			<Box bxsh='1' p='40' bdrs='m'>Box</Box>
			<Box bxsh='4' p='40' bdrs='m'>Box</Box>
		</Flex>
	</PreviewArea>
	<PreviewCode slot='tab' label='JSX'>
		```jsx "is--resetShVar"
		<Flex className='is--resetShVar -g:40' style={{'--sh-hsl': '200 50% 50%', '--sh-a':'20%'}}>
			<Box bxsh='1' p='40' bdrs='m'>Box</Box>
			<Box bxsh='4' p='40' bdrs='m'>Box</Box>
		</Flex>
		```
	</PreviewCode>
	<PreviewCode slot='tab' label='HTML'>
		```html "is--resetShVar"
		<div class="is--resetShVar l--flex" style="--sh-hsl:200 50% 50%;--sh-a:20%">
			<div class="l--box -bxsh:1 -p:40 -bdrs:m">Box</div>
			<div class="l--box -bxsh:4 -p:40 -bdrs:m">Box</div>
		</div>
		```
	</PreviewCode>
</Preview>
